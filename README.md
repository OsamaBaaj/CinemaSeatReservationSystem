# Cinema Seat Reservation System

This is a simple yet interactive C# Windows Forms application for managing cinema seat reservations. It demonstrates:
- Object-Oriented Programming concepts
- Event-driven UI interactions
- Dynamic seat status management (Available, Taken, Reserved, Booked)
- Real-time total ticket price calculation

## Features
- Click to reserve seats
- Hover to preview ticket price
- Confirm or cancel reservations
- Clean and intuitive user interface

Perfect as a learning project or for demonstrating basic desktop application design.

Enjoy testing it and feel free to contribute!
